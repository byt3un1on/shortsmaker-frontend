# ShortsMaker Frontend

Interface web para o sistema ShortsMaker, desenvolvida em Vanilla TypeScript.

## Estrutura do Projeto

- `app/src`: Código fonte TypeScript.
- `app/index.html`: Ponto de entrada da aplicação.
- `app/dist`: Código compilado (gerado pelo build).

## Desenvolvimento

Para iniciar o desenvolvimento, abra este projeto no **VS Code DevContainer**.

### Comandos Úteis

- `make install`: Instala as dependências (npm).
- `make run`: Inicia o servidor de desenvolvimento.
- `make test`: Executa os testes com Jest.
- `make build`: Compila o TypeScript para JavaScript.

## Referência

Este projeto foi baseado no padrão do repositório `byt3un1on/vanilla-site`.
